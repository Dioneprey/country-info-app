# the Country Info App - Dionatan

## Some of the technologies used

### Next.js
- TanStack Query
- TailwindCSS

### Nest.js
- Nest.js
- Redis
- SOLID


## Setup

### Nest.js

Start Docker containers:
```
docker compose up -d
```

Install dependencies:
```
npm i
```

Fill in the .env file (You can find the example in .env.example)

Start the project:
```
npm run start:dev 
```

### React

Install dependencies:
```
npm i
```

Fill in the .env file (You can find the example in .env.example)

Start the project:
```
npm run dev 
```
